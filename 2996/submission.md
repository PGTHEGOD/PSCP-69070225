## STEP 1 อธิบายโจทย์

**โจทย์**
รับคำภาษาอังกฤษยาว 5 ตัว แล้วพิมพ์กลับด้าน คือเอาตัวสุดท้ายมาไว้หน้าสุด ตัวรองสุดท้ายมาเป็นตัวที่สอง ไล่แบบนี้ไปจนครบ แล้วผลลัพธ์ต้องเป็นตัวพิมพ์เล็กทั้งหมด

**Input**
ข้อความตัวอักษรอังกฤษ 5 ตัว 1 บรรทัด

**Output**
ข้อความที่กลับด้านแล้ว เป็นตัวพิมพ์เล็ก 1 บรรทัด

**Constraints**
- ยาว 5 ตัวอักษร
- Timeout 1 วินาที
- Memory 32000 bytes

## STEP 2 สิ่งที่สังเกต

อ่านโจทย์รอบแรกผมงงกับคำว่า "อักขระสับเปลี่ยน" อยู่แป๊บนึง แต่พออ่านซ้ำก็เข้าใจว่ามันคือการอ่านย้อนจากหลังมาหน้า ไม่ได้สลับแบบสุ่มหรือสลับเป็นคู่

ที่ต้องระวังคือเรื่องตัวพิมพ์ ถ้ากลับด้านอย่างเดียวแล้ว input เป็นตัวใหญ่ ผลลัพธ์ก็จะออกมาเป็นตัวใหญ่ ซึ่งผิดโจทย์ ต้องแปลงเป็นตัวเล็กด้วย

edge case ที่นึกออก
- input เป็นตัวใหญ่หมด
- input ผสมตัวใหญ่ตัวเล็ก
- คำที่กลับแล้วเหมือนเดิม เช่น level
- อาจมี space หรือ newline ติดมาท้ายบรรทัด

ความยาวคงที่ที่ 5 ตัว เลยไม่ต้องเช็กความยาวอะไรมาก

## STEP 4 วิธีที่ใช้จริง

ตอนแรกผมคิดว่าจะวนลูปย้อนจากตัวท้ายมาตัวหน้าแล้วต่อ string ไปเรื่อย ๆ แต่พอจะเขียนจริงก็นึกได้ว่า Python มี slice แบบ `[::-1]` ที่กลับด้านให้เลย เลยเปลี่ยนมาใช้อันนี้แทน

เหตุผลที่เปลี่ยนคือมันสั้นกว่า เขียนผิดยากกว่า และไม่ต้องมานั่งคิด index ให้เวียนหัวว่าจะเริ่มที่ 4 แล้วจบที่ตรงไหน ส่วนเรื่องตัวพิมพ์เล็กผมใช้ `.lower()` ต่อท้ายไปเลยในบรรทัดเดียว

`.strip()` ผมใส่ไว้เพราะกลัวมี newline ติดมา ถ้าไม่ใส่แล้วบังเอิญมี space ท้ายบรรทัด พอกลับด้านแล้ว space มันจะไปโผล่หน้าสุด ซึ่งน่าจะทำให้ตอบผิดแบบงง ๆ

โค้ดสุดท้ายเหลือแค่สองบรรทัด

```python
t = input().strip()
print(t[::-1].lower())
```
